# 统计学——Python实现

## 简介

本项目是对《统计学——Python实现》（贾俊平 编著）一书中的例子和习题的Python实现。

> 本项目**非官方**配套资源，为个人学习统计学和Python时的随手记录，仅供学习参考。

## 项目结构

```bash
.
├── pydata                         # 例子中的数据
├── exercise                       # 习题中的数据
├── 第1章数据与Python语言.ipynb
├── 第2章数据的可视化.ipynb
├── 第3章数据的描述统计量.ipynb
├── 第4章随机变量的概率分布.ipynb
├── 第5章参数估计.ipynb
├── 第6章假设检验.ipynb
├── 第7章类别变量分析.ipynb
├── 第8章方差分析.ipynb
├── 第9章一元线性回归.ipynb
├── 第10章多元线性回归.ipynb
└── README.md                      # 项目说明文件
```
